Green House

An IAT 339 project. Created alongside Charlotte Lin

Green House is a website designed for a gardening store. It mainly sells succulents.

Responsibilities

  • Developing the mobile version
  • Developing the tablet version

Main Process

This project was created using Visual Studio Code in the HTML, CSS, and Javascript language. My team member worked on the initial design on all of the version while I worked on developing the mobile and the tablet version of the website.

Design Challenges

  • Adjusting the layout of the desktop version to the mobile

Developing the Versions

As I was mainly responsible for adjusting the user interface in the mobile and tablet version. I had to ensure that our content would not overflow towards the end of the screen. I also used flexboxes in order to wrap the content within the screen.

Conclusion

Our team succeeded in finisihing all version of the prototypes. Furthermore, the feedback that we received was great. From this project, I've become more proficient in using flexboxes in order to design my webpages in HTML and CSS.

Interactive Elements

In-text link

This element is to direct the users to another section or page of the website by clicking on the highlighted version of the text.

CSS selector:

.text-link

Sample code:

<a href="#" class="text-link">Text Link</a>

The in-text link will change color when it is being hovered. Furthermore, after selection, the in-text link will display a different color upon the user's return to the specified page.

Rendered element:

Text Link

Button

This element is used to direct the users to another section within the page or another webpage.

CSS selector:

.btn

Sample code:

<a href="#" class="btn">Default Button</a>

Upon hovering it, the opacity of the button will change. Furthermore, once it has been selected and the user returns back to it, the element will display a different shade of the color.

Rendered element:

Default Button

Image as a Link

An image that directs the users to another section within the page or another webpage.

CSS selector:

.image-link

Sample code:

<a class="image-link" href="https://www.google.com/"> <img src="img/aloe.jpg" alt = "image of a retro video game"></a>

Upon hovering it, the opacity of the image will change.
